Modal

Dialog composicional para fluxos focados, confirmações e detalhes. Usa contexto React e portal para renderização fora da árvore.

Import

import { Modal } from '@/lib'

Source

src/lib/components/modal.tsx

Preview ao vivo interativo

Clique no botão para abrir o dialog e ver como o componente se comporta.

Uso básico 
<Modal.Root>
  <Modal.Trigger>Abrir modal</Modal.Trigger>
  <Modal.Content>
    <Modal.Header>
      <Modal.Title>Detalhes</Modal.Title>
      <Modal.Description>Descrição do dialog.</Modal.Description>
    </Modal.Header>
    <Modal.Body>Conteúdo</Modal.Body>
  </Modal.Content>
</Modal.Root>

Notas

⌨️ Pressionar Esc fecha o modal automaticamente.

Exemplos

Modo controlado open + onOpenChange 
const [open, setOpen] = useState(false)

<Modal.Root open={open} onOpenChange={setOpen}>
  <Modal.Trigger>Abrir</Modal.Trigger>
  <Modal.Content>
    <Modal.Header>
      <Modal.Title>Controlado externamente</Modal.Title>
    </Modal.Header>
    <Modal.Body>Estado gerenciado pelo pai.</Modal.Body>
    <Modal.Footer>
      <Button onClick={() => setOpen(false)}>Fechar</Button>
    </Modal.Footer>
  </Modal.Content>
</Modal.Root>
Confirmação destrutiva async + loading state 
const [loading, setLoading] = useState(false)

async function handleDelete() {
  setLoading(true)
  await deleteItem(id)
  setLoading(false)
}

<Modal.Root>
  <Modal.Trigger asChild>
    <Button variant="destructive">Excluir</Button>
  </Modal.Trigger>
  <Modal.Content>
    <Modal.Header>
      <Modal.Title>Confirmar exclusão</Modal.Title>
      <Modal.Description>Essa ação não pode ser desfeita.</Modal.Description>
    </Modal.Header>
    <Modal.Footer>
      <Modal.Close asChild>
        <Button variant="ghost">Cancelar</Button>
      </Modal.Close>
      <Button variant="destructive" loading={loading} onClick={handleDelete}>
        Excluir
      </Button>
    </Modal.Footer>
  </Modal.Content>
</Modal.Root>
Trigger customizado asChild com qualquer elemento 
// Qualquer elemento vira trigger sem wrapper extra
<Modal.Trigger asChild>
  <img
    src="/thumbnail.png"
    alt="Ver detalhes"
    style={{ cursor: 'pointer' }}
  />
</Modal.Trigger>

// Ou um item de lista
<Modal.Trigger asChild>
  <DropdownMenu.Item>Editar</DropdownMenu.Item>
</Modal.Trigger>

Partes

Modal.Root

Controla estado e contexto do dialog. Suporta modo controlado e não-controlado via open / onOpenChange.

Modal.Trigger

Elemento que abre o modal. Aceita asChild para compor com elementos customizados.

Modal.Content

Janela renderizada em portal via createPortal. Fecha com click no backdrop ou tecla Esc.

Modal.Close

Fecha o modal dentro da composição. Também aceita asChild.

Modal.Header

Container do título e descrição. Adiciona padding para o botão de fechar não sobrepor o conteúdo.

Modal.Footer

Área de ações alinhada à direita, com gap automático entre os filhos.
